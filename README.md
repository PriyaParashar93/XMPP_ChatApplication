# XMPP Chat Application (Android)

This is a**real-time Chat Application for Android** built using **Kotlin**, **Jetpack Compose**, and **Jetpack Navigation**.  
The project uses the **Smack XMPP library** to connect with an XMPP server for authentication, user listing, and real-time one-to-one messaging.


## Features
- XMPP Login / Authentication
- Fetch and display XMPP user list
- One-to-One Chat
- Real-time message sending & receiving
- Modern UI with Jetpack Compose

## Tech Stack
- **Kotlin**
- **Jetpack Compose**
- **Navigation Component**
- **Smack XMPP Library**

## Requirements
- Android Studio (Latest)
- Minimum SDK: 24+ (recommended)
- XMPP Server - Openfire 


## XMPP Server Setup (Openfire)
1. Install Openfire server
2. Create XMPP users from Openfire Admin Console
3. Note down:
   - Server IP / Domain
   - Port (5222)
   - Admin credentials
